ADDONS

 

Newer versions of Minecraft allow you to have more than one resource pack active at a time, so we decided to create some add-on packs to customize your experience with our texture pack.



X

How to install addon packs

  1. Install this pack as you would normally do
  2. Make sure that the pack is moved above the regular pack, and any other packs you have installed.
  3. More descriptive (screenshots in the final thing): use the smaller triangles to move the pack up/down the list. The higher a pack is in the list, the higher priority it will have when being loaded. This means that the game will load the textures from the highest pack first. If a texture is present in more than one pack, the texture from the highest pack will be used
