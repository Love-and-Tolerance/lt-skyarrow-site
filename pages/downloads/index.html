The Archive
Every version of the pack all in one place. Most are no longer supported in the latest version of the game.

How to install

  1. Download the zip file for the version you want to play.
  2. Open Minecraft and navigate to the resource pack settings.
  3. Click the open resource pack folder button.
  4. Copy the zip file you downloaded into this folder.
  5. Go back into Minecraft and back out of the settings screen.
  6. Go back into the resource pack settings.
  7. Enable the resource pack from the list on the left.

The very latest iterations of our pack can be found on our GitHub.

How to download from GitHub

  1. Click the green button saying Clone or Download and select Download ZIP.
  2. Extract the files and take "Love-and-Tolerance-master" folder from inside the extracted folder and move it to %appdata%\.minecraft\resourcepacks
